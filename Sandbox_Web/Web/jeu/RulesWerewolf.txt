Idées : 
	Cartes avec des obstacles ? (=> cartes customs ?)

Phase de jeu : 
	Phase 1 : Journée
		-> Les armes se rechargent
		-> Vision totale
		-> Voyante reconnait les Loup-Garous quand elle les approches
		
		
	Phase 2 : Nuit
		-> Vision diminuée
		-> Loup-Garous se transforment sur commande
		-> Sorcière peut réssussiter un villageois mort
		
	Phase spéciale : Nuit de Brouillard
		-> Masque gris 
		-> Transparence des Personnages dépend de leur vitesse 

	Phase spéciale : Nuit de Pleine Lune
		-> Masque rouge
		-> Vitesse Loup-Garous * 1.25 
		
Personnages : 

Personnages
	-> Laisse une trainée de sang quand hp<20%

Personnages:Loup-Garous 
	-> Temps de transformation : 2 secondes / 0 secondes
		-> Petite taches de sang
	-> [Passif] Mange les non-Loup-Garous
		-> 0,5 secondes de dégat pour tuer
		-> Laisse un corps + trainée de sang dans la direction de fuite
	-> [Passif] Reconnait les Loup-Garous
	-> [Actif] Bondit en avant sur 90 px
	-> Vision de nuit : 300 px
	
Loup-Garous :Grand Loup-Garous 
	-> Vitesse x1.25
	-> Temps de transformation : 0,5 secondes / 0 secondes
	-> Vision de nuit : Totale
	
Personnages:Villageois
	-> [Actif] Tire
		-> Inflige des dégats
		-> Repousse
		-> Villageois qui tire est ralentit
		-> Vision de nuit : 100 px
		
Villageois:Voyante
	-> [Passif] Reconnait les Loup-Garous
	
Villageois:Sorcière 
	-> [Passif] Réssussite un villageois
	
Villageois:Chasseur 
	-> [Actif] 2 balles de fusils
		-> Inflige des degats (1/2 loup || 1 villageois)
		-> Repousse
	
Villageois:Trappeur 
	-> [Actif] 3 pièges
		-> Ralentissement de 60% pendant 1.5 secondes
		-> Visible à 50 px de distance
